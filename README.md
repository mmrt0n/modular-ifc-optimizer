# 모듈러 건축 IFC 최적화 시스템

> IFC 파일에서 벽체 정보를 자동 추출하여 석고보드·합판 시공 배치를 최적화하고,  
> 결과를 인터랙티브 시뮬레이터로 출력하는 자동화 도구입니다.

<br>

## ⬇️ 다운로드 (설치 없이 바로 실행)

| 파일 | 설명 |
|------|------|
| **[IFC최적화.exe](dist/IFC최적화.exe)** | Windows 64비트 실행 파일 (~55MB) |

> Python, Node.js 등 별도 설치 불필요 — EXE 하나로 실행 가능  
> Windows 10/11 64비트 / Chrome·Edge 브라우저 권장

<br>

## ✨ 주요 기능

| 기능 | 설명 |
|------|------|
| 🔍 IFC 자동 파싱 | IfcWall 추출, 치수·개구부 자동 인식 및 단위 보정 |
| 📐 RTL 배치 최적화 | M3 시공방식 — 오른쪽 끝부터 배치, 자투리는 벽 가장자리로 |
| ⚡ 4조합 사전계산 | 석고보드 1P/2P · 합판 1P/2P 모두 EXE에서 사전계산, HTML에서 실시간 전환 |
| ♻️ 재사용 로직 | 자투리 보드 자동 저장 → 같은 공간·층 내 재활용 (ReusePool) |
| 🖥️ 인터랙티브 시뮬레이터 | Python 사전계산 결과 주입, 단계별 애니메이션, 재생 속도 조절 |
| 📊 자재 발주 일람표 | 벽별 수량 집계, 팔레트 단위 발주량 산출 |
| 🔎 QA 자동 전수조사 | 모든 벽 공백·겹침·타공누락·데이터오류 자동 감지 후 화면 상단 경보 |

<br>

## 🎨 시뮬레이터 색상 가이드

| 색상 | 종류 | 설명 |
|------|------|------|
| 🟢 초록 | 온장 (full) | 규격 그대로 부착 (석고보드 900×1800 / 합판 1220×2440) |
| 🟠 주황 | 직선절단 (edge_cut) | 벽 가장자리에 맞게 폭·높이를 직선으로 자른 보드 |
| 🟣 보라 | 노치절단 (notch) | 개구부 모서리 ㄱ/ㄴ형 절단 |
| 🔵 파랑 | 재사용 (reuse) | 다른 벽 자투리를 재활용한 보드 |

<br>

## 🚀 사용 방법

1. `IFC최적화.exe` 실행
2. **자투리 재사용** 여부 선택 (활성 권장)
3. **IFC 파일** 선택
4. (선택) 교차검증 기준값 입력
5. ▶ 실행 → 브라우저에서 결과 자동 오픈

> **자재(석고보드/합판)·공법(1P/2P)** 은 EXE 실행 시 선택하지 않습니다.  
> 시뮬레이터 HTML 안에서 버튼으로 언제든 전환 가능합니다.

**출력 파일 (IFC 파일과 같은 폴더에 저장)**

```
프로젝트_검증보고서.html              ← IFC 벽체 검증 리포트
프로젝트_석고보드_2P최적화결과.html   ← 배치도 + 자재 집계 (기본 뷰)
프로젝트_시뮬레이터.html             ← 인터랙티브 시뮬레이터 ★
```

## 🛠️ 개발 환경 설정 (소스 수정 시)

```bash
git clone https://github.com/mmrt0n/modular-ifc-optimizer.git
cd modular-ifc-optimizer
pip install ifcopenshell numpy pyinstaller
python ifc_pipeline.py
```

**EXE 재빌드**

```bash
# 반드시 프로젝트 루트 디렉터리에서 실행
python -m PyInstaller IFC최적화.spec --clean --noconfirm
# 결과물: dist/IFC최적화.exe
```

## 📋 시공 로직 구현 현황 (M3 방식)

| 항목 | 구현 여부 | 비고 |
|------|-----------|------|
| RTL (오른쪽 끝부터) 배치 | ✅ | Python EXE 사전계산 |
| 자투리 벽 가장자리 배치 | ✅ | 개구부 옆 작은조각 제거 |
| 개구부 중심 대칭 (SYM) | ❌ | RTL 가장자리 배치로 대체 |
| 각재 피치 450mm (STUD) | ✅ | 2P 레이어 엇갈림 |
| 바닥 밀착 (FLOOR_GAP=0) | ✅ | M3 확인결과 9-1 |
| 직선절단 / 노치절단 분류 | ✅ | edge_cut / notch |
| 재사용 풀 (MIN_W=300 / MIN_H=450) | ✅ | Python 엔진 |
| 재사용 풀 leftover 처리 | ✅ | 사용 후 2조각 풀 재저장 |
| 재사용 보드 시각화 🔵 | ✅ | |
| 4조합 동시 사전계산 | ✅ | gyp1/gyp2/ply1/ply2 → PRECOMPUTED_ALL |
| 합판 팔레트 100매 | ✅ | 전체 통일 |
| QA 자동 전수조사 | ✅ | 공백·겹침·타공누락·코드중복 4종 |
| SVG 도면 스케일 보정 | ✅ | 좁은 벽 최솟값 강제 시 비율 보정 |
