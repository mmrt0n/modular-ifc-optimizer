# 모듈러 건축 IFC 최적화 시스템

> IFC 파일에서 벽체 정보를 자동 추출하여 석고보드·합판 시공 배치를 최적화하고,  
> 결과를 인터랙티브 시뮬레이터로 출력하는 자동화 도구입니다.

<br>

## ⬇️ 다운로드 (설치 없이 바로 실행)

| 파일 | 설명 |
|------|------|
| **[IFC최적화_v5.exe](dist/IFC최적화_v5.exe)** | Windows 64비트 실행 파일 (~54MB) |

> Python, Node.js 등 별도 설치 불필요 — EXE 하나로 실행 가능  
> Windows 10/11 64비트 / Chrome·Edge 브라우저 권장

<br>

## ✨ 주요 기능

| 기능 | 설명 |
|------|------|
| 🔍 IFC 자동 파싱 | IfcWall 추출, 치수·개구부 자동 인식 및 단위 보정 |
| 📐 다중 개구부 배치 | 벽당 문+창 여러 개를 각각 독립적으로 정확히 처리 |
| ⚡ 2조합 사전계산 | 석고보드 1P · 합판 1P 사전계산, 시뮬레이터에서 실시간 전환 |
| 📐 시공 방향 | 세로(종방향) 고정 — M3 표준 (3-1) |
| 📈 할증률 적용 | 발주 수량에 할증률(%) 자동 계산 후 최적화 결과에 표시 |
| ♻️ 재사용 로직 | 자투리 보드 자동 저장 → 같은 공간·층 내 재활용 (ReusePool) |
| 🖥️ 자체완결형 시뮬레이터 | 단계별 애니메이션, 석고보드·합판 전환, 합판보강 전환, 재생 속도 조절 |
| 📊 자재 발주 일람표 | 벽별 수량 집계, 팔레트 단위 발주량 산출 |
| 🔎 QA 자동 전수조사 | 모든 벽 공백·겹침·타공누락·데이터오류 자동 감지 후 경보 |

<br>

## 🎨 시뮬레이터 색상 가이드

| 색상 | 종류 | 설명 |
|------|------|------|
| 진파랑 | 온장 | 규격 그대로 부착 (석고보드 900×1800 / 합판 1220×2440) |
| 연파랑 | 절단 | 벽 가장자리·개구부에 맞게 자른 보드 |
| 노랑 | 개구부 처리 (cut_op) | 창문 위·아래 구간 보드. 위+아래 합이 1장 이내면 원판 1장에서 재단 |
| 초록 | ♻ 재사용 | 다른 벽 자투리를 재활용한 보드 |
| 회색 | 개구부 | 문·창 공간 |

## 🚀 사용 방법

1. `IFC최적화_v5.exe` 실행
2. **자재** 선택: 석고보드 / 합판
3. 시공 방향은 **세로(종방향) 고정** (M3 표준)
4. **할증률(%)** 입력 (선택, 기본값 0)
5. **자투리 재사용** 여부 선택 (활성 권장)
6. **IFC 파일** 선택
7. (선택) 교차검증 기준값 입력
8. ▶ 실행 → 브라우저에서 결과 자동 오픈

**출력 파일 (IFC 파일과 같은 폴더에 저장)**

```
프로젝트_검증보고서.html                  ← IFC 벽체 검증 리포트
프로젝트_석고보드_1P최적화결과.html       ← 배치도 + 자재 집계 (선택 자재 기준)
프로젝트_시뮬레이터.html                 ← 인터랙티브 시뮬레이터 ★
```

## 🛠️ 개발 환경 설정 (소스 수정 시)

```bash
git clone https://github.com/mmrt0n/modular-ifc-optimizer.git
cd modular-ifc-optimizer
pip install ifcopenshell numpy pyinstaller
py -3 ifc_pipeline.py
```

**EXE 재빌드**

```bash
py -3 -m PyInstaller IFC최적화_v5.spec --clean --noconfirm
# 결과물: dist/IFC최적화_v5.exe
```

## 📋 시공 로직 구현 현황 (M3 방식)

| 항목 | 구현 여부 | 비고 |
|------|-----------|------|
| 배치 방향 (RTL / LTR) | ✅ | **개구부 없는 벽=RTL(오른쪽 끝부터, 8-1) / 개구부 있는 벽=LTR(왼쪽 하단부터 온장, 9번)** |
| 자투리 벽 가장자리 배치 | ✅ | 개구부 없는 벽 자투리 왼쪽, 개구부 벽 자투리 개구부·오른쪽 |
| **다중 개구부 독립 처리** | ✅ | **문+창 혼합 벽 각각 정확 처리** |
| 각재 피치 450mm (STUD) | ✅ | 1P 단겹 시공 |
| 바닥 밀착 (FLOOR_GAP=0) | ✅ | M3 확인결과 9-1 |
| 직선절단 / 개구부처리 분류 | ✅ | cut / cut_op |
| 재사용 풀 (MIN_W=300 / MIN_H=450) | ✅ | ReusePool |
| **시공 방향** | ✅ | **세로(종방향) 고정 — M3 표준 (3-1)** |
| **할증률(%) 발주 계산** | ✅ | **최적화결과 HTML 할증 발주량 카드 표시** |
| 2조합 사전계산 | ✅ | gyp1(석고보드) / ply1(합판) — 1P 고정 |
| 합판 보강 영역 계산 | ✅ | 도어·TV·선반 보강 자동 산출 |
| QA 자동 전수조사 | ✅ | 공백·겹침·타공누락·코드중복 4종 |
| SVG 도면 스케일 보정 | ✅ | 좁은 벽 비율 보정 |
| **자체완결형 시뮬레이터** | ✅ | **템플릿 주입 방식 폐기, 단일 HTML 생성** |
