단순 전체 RTL (기존 방식 문제)
  • 개구부 무시 → 왼쪽 끝에 자투리 1개
  • 개구부 사이 구역마다 잔 조각 추가 발생
  • 5개 개구부 → 잔 조각 5+ 개 산발
구역별 독립 처리 (적용)
  • 양쪽 끝 구역 — RTL: 자투리 벽 가장자리
  • 개구부 사이 구역 — LTR: 온장이 이전 개구부에 밀착, 자투리 다음 개구부 쪽
  • 각 구역 독립 계산 → 조각 집중·최소화

_columns_rtl() · _columns_ltr() — 각 구역을 독립 리스트로 계산 후 합산  ·  _fix_thin_edge_columns() — 300mm 미만 잔 조각 보정
